講義怎麼學

這不是備註清單，而是把考點整理成能吸收的路線。先看骨架，再看怎麼考，最後用表格和題庫驗證。

先主線 再比較 最後刷題

1. 先看骨架

先從風險管理、精算與準備金、商品與服務、法規與合規四條主線下手。法規不要單條硬背，請先記成：投保前 → 存續中 → 事故後 → 合規控管。

2. 再看題型

題目通常是定義、比較、流程或數字四種。看到新名詞時，先判斷它是在考概念、差異、順序，還是考數字。

風險管理主軸

先掌握分類，再掌握衡量與管理。這部分是整份題庫最基礎、也最容易反覆出題的骨架；不是背一串名詞，而是看懂分類邏輯。

客觀 / 主觀 純粹 / 投機 基本 / 特定

先分清楚：四種分類維度

風險不是只分一種。題庫最常考的是先確認你在看哪個維度：是否可衡量、損失性質、事故原因、影響範圍。

分類內容

  • 是否可衡量：客觀風險與主觀風險。
  • 損失性質：投機風險與純粹風險。
  • 事故原因：靜態風險與動態風險。
  • 影響範圍：基本風險與特定風險。

怎麼衡量

風險衡量看頻率與幅度，損失成本 = 頻率預期值 × 幅度預期值。這句不是公式裝飾，而是題目在問你『怎麼估算損失』。

怎麼管理

風險管理四字：避免、預防、自留、移轉；保險屬於移轉。題目若在問工具，先想是分散、降低還是轉嫁。

一眼記法

投機風險看獲利可能，純粹風險只看損失機會。特定風險範圍小、較容易控制。

出題方式

題庫常先用一個分類詞迷惑你，再換另一個分類詞問同一件事。

考題陷阱

把可衡量和性質混在一起，或把分類和衡量混在一起，是最常見的失分點。

精算與準備金

這一段負責把「保費、利率、生命表、準備金」串成一條鏈。題目常以不同名詞包裝，但邏輯一致。

生命表 預定利率 責任準備金

生命表 / 經驗生命表

生命表用來推估死亡率與存活率。經驗生命表則是依歷史統計或實際保戶經驗編製，常用於費率與準備金。

費率設計

壽險保費的核心原則是收支相等與互助分攤，預定危險發生率、預定利率與預定附加費用率都會影響費率。

責任準備金 / 保單價值準備金

責任準備金是為未來契約責任提存；保單價值準備金則與契約累積價值、解約與借款密切相關。

年金 / 壽險 / 健康

年金看生存條件；壽險看死亡給付；健康保險看疾病與傷害。這三者是商品分類的核心分野。

傳統型年金

傳統型與利率變動型年金都會用到年金生命表、預定危險發生率與責任準備金。

高死低利

責任準備金提存通常採較高死亡率、較低利率的保守假設。

保單管理

保費自動墊繳、保單借款與保單活化，都是保單價值準備金常見的應用情境。

商品與服務

商品不是單獨看名稱，而是看給付條件、適用族群與風險控管。這裡也包含保險科技與售後服務。

健康 / 長照 團體 / 學生 投資型 / 科技

健康保險

補償因疾病或傷害造成的醫療支出與工作損失。常與醫療、失能、住院、實支實付等概念一起考。

長期照顧保險

重點在生活自理與認知障礙是否達到約定門檻。題庫常考 ADL、CDR 與免責期間。

團體保險 / 學生團體保險

團體保險常依團體風險、薪資或職務決定費率。學生團體保險則要注意主管機關、專戶與給付項目。

投資型保險 / 外溢效果 / 電子保單

投資型保險連結投資標的；外溢效果透過行為回饋改善健康；電子保單與保險存摺則強化查詢與驗證。

業務員售後服務

不是交保單就結束，而是協助客戶做保額調整、附約補強、保費規劃與保單活化。

保單科技平台

保險存摺、電子保單、身分驗證與理賠協助，構成現在的保險科技服務鏈。

人身風險對家庭的影響

醫療費用增加、收入減少、遺族生活費用增加與退休規劃，是實務規劃最常見的四項。

金融市場常識補強

把第三份題庫整理成可讀的知識鍊：先看市場結構，再看監理分工，最後落到保險商品、消費者保護與合規。

市場結構 監理合規 保險與消保

1. 金融市場骨架

金融市場的核心，是把資金從供給者導向需求者。看到題目時，先判斷它在考『功能』、『工具』、『市場層次』，還是『監理單位』。這一區最常混淆的是資本市場、貨幣市場、初級市場、次級市場與直接金融、間接金融。

2. 監理與分工

  • 金管會是金融監理總樞紐，銀行、證券、保險都會考它的角色。
  • 證券、保險、銀行常被拆成不同局處或業務範圍來考。
  • 遇到『檢查業務』、『法規訂定』、『統籌監理』，先找主管機關。
  • 題目如果問『哪一個不是隸屬金管會』，要先排除名稱相近但權責不同的機構。

3. 保險市場骨架

  • 傳統保險看保障功能，投資型保險看保障與投資雙重功能。
  • 人身保險、財產保險、責任保險、健康保險與年金保險常互相比較。
  • 弱勢族群與普惠金融常連到微型保險、招攬原則與商品適配性。
  • 保險題常問的不是『名詞定義』，就是『誰可以承保、賠什麼、誰主管』。

4. 消費者保護與合規

  • KYC 是先認識客戶，再談商品；AML/CFT 是防堵異常交易與資恐風險。
  • STR、CTR、PEP 常一起出現，重點在申報、監控與風險辨識。
  • 金融消費爭議處理要會認評議中心，不要把公會或業務單位混進去。
  • 公平待客、廣告真實、適合度與不歧視，常是情境題的答案線索。

金融市場常識知識鍊

考前快速複習時，請照這條順序看：市場功能 → 監理分工 → 保險商品 → 消費者保護 → 合規申報。

市場層

資金供需 → 金融工具 → 初級市場 / 次級市場 → 資本市場 / 貨幣市場

監理層

金管會統籌 → 銀行 / 證券 / 保險分工 → 檢查與法規執行

合規層

KYC → AML/CFT → STR / CTR → PEP 與高風險客戶

保險層

傳統保險 → 投資型保險 → 微型保險 → 弱勢族群與普惠金融

法規與合規

這一區把保險法、金融消保、個資與洗錢防制串成一條線。題目常考定義、效力與例外；真正的記法應該是投保前、存續中、事故後、爭議時與合規時各看什麼。

當事人 告知 / 通知 受益人規則
投保前 存續中 理賠 爭議 監理

契約當事人

保險人與要保人是當事人；被保險人與受益人是關係人。要保人要有保險利益與行為能力。

保險利益

保險利益是人身保險合法性的核心。本人、家屬與生活費或教育費所仰給之人，是常見的可保對象。

告知 / 通知 / 最大誠信

訂約時要告知重要事項；事故或危險增加時要通知保險人；保險人與業務員也要誠實處理資訊。

受益人與消保

受益人有疑義時，推定要保人。條款有疑義時作有利被保險人的解釋，顯失公平條款則該部分無效。

個資 / 洗錢 / 電銷

個資蒐集、特種個資、保存年限、可疑交易申報與電話行銷規範，都是近年高頻合規題。

定型化契約

保險契約多屬定型化契約，企業經營者應提供合理審閱期，且條款有疑義時要有利消費者解釋。

保險業組織

保險公司、合作社、代理人、經紀人與授權代收保費的界線要分清楚，容易混題。

投保前知識鍊

需求分析 → 保險利益 → 要保書 → 告知義務 → 核保 → 承保。題目若在問契約怎麼開始，先回到這條鏈。

契約存續知識鍊

繳費 → 寬限期 → 危險增加通知 → 契約變更 / 保單借款 / 自動墊繳 → 續存或解約。

理賠知識鍊

事故發生 → 通知保險人 → 申請文件 → 調查審查 → 給付 / 拒賠。這條鏈常跟時效一起考。

合規知識鍊

KYC → 個資告知 → 大額交易確認 → 可疑交易申報 → 內控稽核。這條鏈是保險業常見的監理主線。

法規主線 1：成立與效力

投保前先看保險利益與告知義務；保單成立後再看契約條款、繳費與生效。考題常把「可否投保」和「何時生效」混在一起。

法規主線 2：存續與變更

契約存續時，重點是危險增加通知、保費給付、寬限期、保單借款、自動墊繳與契約變更。看到『存續中』三個字，就回這條線。

法規主線 3：理賠與請求

事故發生後先通知，再備文件、等審查，最後才是給付或拒賠。若題目涉及請求權、文件與時效，先從這條線找答案。

法規主線 4：爭議與監理

條款有疑義時作有利被保險人的解釋；若牽涉個資、洗錢或銷售行為，再回到金融消保與合規監理。這是考前最常被忽略的一段。

法規主線 5：銷售與說明

行銷話術、保險商品說明書、重要事項告知、審閱與簽收，常一起出現在考題裡。看到『銷售』，先想資訊是否充分、是否誤導。

法規主線 6：個資與資料使用

蒐集、處理、利用個人資料前，先看告知目的、利用範圍與保存管理。保險業常把個資題和申訴、電銷、核保一起考。

法規主線 7：洗錢防制

認識客戶、風險評估、大額交易確認、可疑交易申報、持續監控，構成保險業的 AML 主線。考題常把門檻、對象與申報期限混在一起。

法規主線 8：申訴與評議

客訴、申訴、調處與評議是爭議處理的終點線。先看內部申訴，再看外部救濟，通常比硬背機關名稱更好記。

投保前

投保前先看保險利益，再看告知義務與要保書內容，最後才進到核保與承保。

題型提示：常把「可否投保」與「何時生效」混成一題，先把成立條件與生效條件分開。

存續中

契約存續時，重點轉到繳費、寬限期、危險增加通知、保單借款與自動墊繳。

題型提示：只要題目出現『存續中』、『變更』或『借款』，就回到這條鏈。

理賠

事故發生後先通知保險人，再準備理賠文件、配合審查，最後才是給付或拒賠。

題型提示：常把通知時點、文件齊備與給付期限混在一起考，順序別倒。

爭議

條款疑義時應作有利被保險人的解釋；涉及個資、洗錢或銷售爭議時，回到金融消保與監理規則。

題型提示：凡是『條款有疑義』、『顯失公平』、『申訴評議』，都先看消費者保護方向。

監理

監理與稽核重點放在內控、可疑交易申報、個資保護與銷售合規；題目若問主管機關職掌，就回到 FSC / 保險局 / 檢查局 / 洗防辦。

題型提示：只要題目有『監理』、『稽核』、『申報』、『裁罰』，就先想是否屬於外部監理或內部控制。

生存保險 / 死亡保險 / 生死合險前者活著給付，後者死亡給付，生死合險兩者兼具看給付條件

財產保險 / 責任保險前者保財物損失，後者保法律責任看賠的是物還是責任
要保人 / 被保險人 / 受益人前者簽約與繳費，中者是風險標的，後者領取保險金三者角色不同
